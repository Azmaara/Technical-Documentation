What is jquery?

jquery is a lightweight, "write less, do more", JavaScript library.

The purpose of jQuery is to make it much easier to use JavaScript on your website.

jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.

jQuery also simplifies a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.

The jQuery library contains the following features:

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX
  • Utilities

jQuery hide()

With jQuery, you can hide and show HTML elements with the hide() methods:

            $("#hide").click(function(){
            
              $("p").hide();
              
            });

jquery show()

With jquery, you can hide and show HTML elements with the show() methods:

            $("#show").click(function(){
            
              $("p").show();
              
            });
jquery HTML Reference

Three simple, but useful, jQuery methods for DOM manipulation are:

  • text() - Sets or returns the text content of selected elements
  • html() - Sets or returns the content of selected elements (including HTML markup)
  • val() - Sets or returns the value of form fields

jQuery text()

The following example demonstrates how to get content with the jQuery text() methods:

            $("#btn1").click(function(){
            
              alert("Text: " + $("#test").text());
              
            });

jquery html()
